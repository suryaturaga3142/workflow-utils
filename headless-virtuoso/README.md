## Headless Virtuoso

The idea of this project is to automate _some_ parts of working with analog IC design. Stuff like lib and cell creation, keeping files neat, autoplacing standard parts, etc. This project does it really well, but it is still in its early stages. The goal isn't to replace the GUI, since circuits need that. 

The next steps would be tough, I'd wanna automate things like running long simulations. The complicated part is passing parameters and stuff, so I doubt it's worth it. I'll do what I can when my courses give me access to Virtuoso.
